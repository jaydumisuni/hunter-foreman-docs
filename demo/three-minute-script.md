# Three-Minute Demo Script

## 0:00 — Problem

Small businesses receive requests from everywhere. The issue is not only getting messages. The issue is turning those messages into owned work.

## 0:25 — Introduce Hunter Foreman

Hunter Foreman is an AI operations foreman. ROSE acts as the receptionist. Foreman turns the request into a task, chooses the workflow, and keeps humans in control.

## 0:45 — Show ROSE Intake

Open the main Hunter Foreman app.

Use a demo request:

```text
We need event booking, digital invitations, guest tickets, QR check-in and a clear approval flow.
```

Submit the request.

## 1:15 — Show Foreman Routing

Point out:

- intent
- confidence
- workflow
- owner/team
- escalation status
- notification previews

## 1:45 — Show App Bridge

Open the receiver app.

Show that the created task was dispatched to the connected app through the versioned bridge contract.

Point out:

- contract version
- event type
- task ID
- ROSE → Foreman → AppBridge timeline

## 2:20 — Explain Public Safety

This demo is public-safe. It does not expose private Hunter internals, production credentials, private prompts, client data, or repair/recovery modules.

## 2:40 — Close With Vision

The vision is a business-owned AI operations layer that can run in the cloud or locally on an AMD Ryzen mini PC for always-on automation.

Hunter Foreman is the first public extraction from the larger Hunter ecosystem.
